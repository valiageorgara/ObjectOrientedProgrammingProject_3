ΕΝΤΟΛΗ ΜΕΤΑΓΛΩΤΤΙΣΗΣ:
			make
ΕΝΤΟΛΗ ΕΚΤΕΛΕΣΗΣ:
			exe 1ο_όρισμα 2ο_όρισμα

όπου 1ο_όρισμα=Μ και 2ο_όρισμα=Ν

ΠΡΟΤΕΙΝΕΤΑΙ: Να μπουν σε σχόλια τα μηνύματα των destructors για να φαίνονται καλύτερα τα υπόλοιπα μηνύματα.
Για να φαίνεται καλύτερα η ποσοστιαία μεταβολή ΠΡΟΤΕΙΝΟΝΤΑΙ μεγάλοι αριθμοί ορισμάτων, αλλιώς κάποιο σωματίδιο έχει παράγει 0, 
οπότε βγαίνει -1 το ποσοστό που σημαίνει ότι δεν ορίζεται η μεταβολή!
Η υλοποίηση του 3ου ερωτήματος χωρίζεται σε 2 περιπτώσεις: αν τα 2 σωματίδια που έχω αλλάξει βρίσκονται στο τέλος του vector ή οπυδήποτε αλλού.
